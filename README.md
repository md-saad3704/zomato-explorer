# Zomato India Restaurant Explorer

Interactive multi-city analytics dashboard built using:

- Streamlit
- Pandas
- Plotly

## Features

- City-based analytics
- Cuisine insights
- Locality analysis
- Hidden gems finder
- City comparison
- Smart search
- Data transparency system

## Dataset

Zomato India Restaurants Dataset (2019)

## Disclaimer

Restaurant information may have changed since data collection.
Always verify before visiting.
